# Basic Starter Template
**Basic starter for a HTML, SCSS, JS project**

Yarn should be installed.

`.gitignore` ignores:
- `/node_modules`
- `yarn-error.log`

Instructions:

Run `yarn install`

Installs:

- **autoprefixer**
- **browser-sync**
- **node-sass**
- **onchange**
- **parallelshell**
- **postcss-cli**

Run `yarn startapp`

Starts browser-sync server with hot reload, runs sass compiler and autoprefixer
